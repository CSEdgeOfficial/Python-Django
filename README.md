## Quick Start

1. **Install Django**
   ```bash
   pip install django
   ```

2. **Apply Migrations**
   ```bash
   python manage.py migrate
   ```

3. **Run Server**
   ```bash
   python manage.py runserver
   ```

4. **Visit**: `http://127.0.0.1:8000/`
